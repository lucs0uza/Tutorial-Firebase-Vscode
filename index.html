Essential Terminal Commands in VS Code

Now that you've configured all your environment variables, let's explore the commands you can use in VS Code's integrated terminal to boost your productivity.

LS
Lucas de Souza
December 14, 2025
8 min read

Now that you've configured all your environment variables, let's explore one of VS Code's most powerful features: the Integrated Terminal. VS Code provides a full-featured terminal directly within the editor, allowing you to run commands without switching between windows.

Terminal 
cd Desktop
Tip

VS Code terminal automatically detects your system's default shell, but you can configure it to use PowerShell, CMD, Git Bash, WSL, or any other shell you prefer!

Why Use the VS Code Integrated Terminal?

VS Code's integrated terminal offers several advantages:

  • Seamless workflow: No need to switch between windows ‚Äî run commands directly where you code.
  • Multiple terminals: You can have multiple terminals open simultaneously, each with its own session and shell.
  • Environment aware: The terminal inherits your configured environment variables and workspace settings.
  • Split terminals: Split your terminal panel to run multiple commands side by side.

Basic Terminal Navigation

Before diving into development, it's essential to master navigation commands. These are the ones you'll use most often:

Terminal 
cd folder-name       # Enter a folder
cd ..                # Go back one folder
cd ~                 # Go to home folder
pwd                  # Show current directory
clear                # Clear the terminal

Listing and Manipulating Files

Knowing how to list and manipulate files via terminal is fundamental for any developer:

Terminal 
ls                   # List files and folders
ls -la               # List with details and hidden files
ls -lh               # List with readable sizes
tree                 # Show tree structure
Terminal 
mkdir new-folder     # Create a new folder
touch file.txt       # Create an empty file
rm file.txt          # Remove a file
rm -rf folder        # Remove folder and contents
cp source dest       # Copy file
mv source dest       # Move or rename

Essential Git Commands

VS Code has excellent Git integration, but knowing terminal commands gives you full control over version control:

Terminal 
git status                   # Check file status
git add .                    # Add all files
git commit -m "message"      # Make a commit
git push origin main         # Push to remote repository
git pull origin main         # Pull updates
git branch                   # List branches
git checkout -b new-branch   # Create and switch to new branch
git log --oneline            # Summarized history
git diff                     # View uncommitted changes
VS Code Git Integration

VS Code's Source Control panel provides a visual interface for Git, but the terminal gives you access to advanced commands and scripts that aren't available in the GUI!

Managing Packages and Dependencies

Depending on your stack, you'll use different package managers. Here are the most common ones:

Node.js / JavaScript:

Terminal 
npm install                  # Install dependencies
npm install package          # Install a package
npm run dev                  # Run development script
npm run build                # Generate production build
npx create-next-app          # Create Next.js project

Python:

Terminal 
pip install package          # Install a package
pip install -r requirements.txt  # Install from file
python -m venv venv          # Create virtual environment
source venv/bin/activate     # Activate environment
python app.py                # Run script

VS Code Terminal Shortcuts

Master these shortcuts to navigate the terminal like a pro:

  • Ctrl + `: Open/close terminal panel
  • Ctrl + Shift + `: Create new terminal instance
  • Ctrl + C: Cancel running command
  • Ctrl + L: Clear terminal screen
  • Tab: Autocomplete commands and paths
  • Arrow ‚Üë / ‚Üì: Navigate command history
  • Ctrl + Shift + 5: Split terminal

Running Development Servers

One of the most common tasks is running development servers. VS Code's terminal makes it easy to start and manage your servers while you code:

Terminal 
# React / Next.js / Vite
npm run dev

# Python Flask
flask run

# Python Django
python manage.py runserver

# Go
go run main.go

# Simple HTTP server (Python)
python -m http.server 8000

Advanced Tips

Some techniques to supercharge your terminal usage in VS Code:

  • Aliases: Create shortcuts for long commands in your shell profile (.bashrc, .zshrc, or PowerShell profile)
  • Pipes: Combine commands with | to filter outputs (e.g., ls | grep .js)
  • Tasks: Configure VS Code tasks in tasks.json to automate common terminal commands
  • Terminal profiles: Set up multiple shell profiles for different project needs
